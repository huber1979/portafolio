## Deploy en Netlify

1. Sube este repositorio a GitHub
2. Entra a [app.netlify.com](https://app.netlify.com)
3. "Add new site" → "Import an existing project"
4. Conecta tu cuenta de GitHub y selecciona este repo
5. Build command: (vacío) · Publish directory: `.` (raíz)
6. Click "Deploy site"

## Deploy en GitHub Pages

1. Renombra el repo a `huber1979.github.io`
2. Settings → Pages → Branch: `main`, folder: `/ (root)`
3. El sitio quedará en: `https://huber1979.github.io`
